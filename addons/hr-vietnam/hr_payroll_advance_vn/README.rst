===============
Advanced Salary
===============

This module allow to make a salary payment in advance.
The advanced amount will be deducted to the salary slip by the end of month.


Installation
============

To install this module, you need to install hr_payroll_vn

Usage
=====

To use this module, you need to:

* Go to Payroll > Advanced Payslips
