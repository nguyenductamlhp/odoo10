=========================
HR Recruitment
=========================

This module extends HR Recruitment to allow:
    - Manage a lot of essential information about applicant.
    - Convert those information from applicant to employee.

Installation
============

To install this module, you need to install the following module:
    - `hr_recruitment`
    - `hr_employee_vn`
